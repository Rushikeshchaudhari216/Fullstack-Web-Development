HTML Code Example

To include a script, use the <script> tag like this:

<script>
  alert("Hello, world!");
</script>

Special Characters

  • Less than: &lt; → <
  • Greater than: &gt; → >
  • Ampersand: &amp; → &
  • Double quote: &quot; → "

Keyboard Shortcut Example

Press Ctrl + S to save your file.
